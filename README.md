# Crypto-Historical-Price-Analysis-Prediction

Descripcion: Desarrolle un modelo de prediccion de valores futuros de criptomonedas que puede ser modificado para seleccionar la cripto que se desee predecir, en este caso se analizo la moneda Ethereum. La metrica de evaluacion que se utilizo para ver que tan bueno era el modelo fue el MAE y el RMSE.

Tecnoligia y librerias utilizadas: Utilice Python y las librerias pandas, sklearn, tensorfow, keras, matplotlib y numpy.

Responsabilidad clave: Se realizo una EDA (analisis exploratorio de datos), se creo un modelos con la libreria keras de tensorflow con datos de entrenamiento y se realizo un test con datos de validacion buscando el valor mas bajo de MAE y RMSE.

Impacto logrado: Se obtuvo un modelo con un valor de MAE de $109 USD y el un valor de RMSE de $147 USD que es aceptable para un mercado tan volatil como el de las criptomonedas
